# OpenBrain

OpenBrain 是一个工作室官网原型项目，目前首页主展示项目为“上海理工大学沉金 PCB 校园卡”。

## 当前内容

- 首屏沉浸式展示：PCB 图纸与实体卡片同位切换，随后进行正面到背面的 3D 翻转。
- 多项目结构：当前只有一个项目，但页面已经预留后续项目卡位。
- 购买信息：包含价格、联系方式，以及抖音/闲鱼链接复制入口。
- 说明区：包含校园卡芯片移植与半替代方案说明。

## 文件结构

- `index.html`：页面结构与文案
- `styles.css`：首页视觉、布局与动画样式
- `main.js`：滚动驱动动效与复制按钮逻辑
- `stage_blueprint.png`：统一舞台坐标下的 PCB 图纸素材
- `stage_front.png`：统一舞台坐标下的卡片正面素材
- `stage_back.png`：统一舞台坐标下的卡片背面素材

## 本地预览

在项目目录运行：

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8000/
```
